dictionaryOfValidationFunctions(input, rules) → {boolean}

Based on a set of rules, is the input valid?
Parameters:
Name Type Description
input
rules If rules is a string, the parameter defines the allowable input type as one of the following: 'boolean', 'string', 'number', 'function'. If rules is an object that contains the key 'arrayOf', then all items in the array will be held to the rule in rules.arrayOf. (Note: No other key-value pairs in rules will be considered in this case) TO BE CODED: If rules is defined as an object, each key of rules must exist in the input, and each key must have the type defined as the value of that key
